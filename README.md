# FileBatchToolbox - 文件批量处理工具箱

一个简单实用的批量文件处理工具，适合个人和小企业日常使用。

## 功能介绍

### 1. 批量重命名
- **顺序编号重命名**: 按顺序添加编号，如 `photo_001.jpg`
- **日期前缀重命名**: 添加日期前缀，如 `20240315_photo.jpg`
- **搜索替换重命名**: 批量替换文件名中的特定文字
- **统一扩展名**: 批量修改文件扩展名

### 2. 图片批量压缩
- 支持 JPG、PNG、BMP、GIF、WebP 等格式
- 可调节压缩质量 (1-100)
- 自动备份原图
- 显示压缩前后对比和节省空间

### 3. 批量文本替换
- 支持普通文本替换
- 支持正则表达式替换
- 支持多种文本格式 (txt, md, json, xml, csv, log, html, css, js, py 等)

## 安装

```bash
# 安装依赖
pip install Pillow

# 运行工具
python file_batch_toolbox.py
```

## 使用方法

### 批量重命名示例

```
选择功能: 1
请输入要处理的目录路径: C:\Users\YourName\Photos

找到 25 个文件
文件列表:
  1. vacation.jpg
  2. beach.png
  ...

是否继续 (y/n): y

输入命名模式 (默认 {name}_{index:03d}): IMG_{index:04d}

```
